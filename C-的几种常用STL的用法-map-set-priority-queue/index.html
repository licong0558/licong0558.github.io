  • set

    • C++的set是利用平衡二叉树实现的一个容器，具有以下特点

      1. 对于插入、删除和查找操作,set保证其时间复杂度都是$O(\log_2n)$（n是set中元素的个数）。
      2. set是一个有序的、可以前向和后向遍历的容器（双向迭代器）。
      3. set的元素可以插入、删除，但是不可更改。
      4. set中的元素不可重复。

      至于具体的原理去学习一下二叉排序树和平衡二叉树就会了解了
